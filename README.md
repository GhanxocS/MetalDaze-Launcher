## ¿Qué es MetalDaze Launcher?

MetalDaze Launcher es el launcher oficial de **MetalDaze Estudio**, diseñado para ofrecer una experiencia de juego simple, rápida y centralizada para los servidores y proyectos de la comunidad.

Desde el launcher podés iniciar sesión con tu cuenta de Microsoft, elegir tu instancia de juego, ver las últimas noticias del servidor y configurar todo sin tocar ningún archivo manualmente.

---

## Características

- 🔐 **Login con Microsoft** — autenticación oficial con cuentas premium
- 🎮 **Múltiples instancias** — cambiá entre distintas versiones o modpacks desde el mismo launcher
- 📰 **Noticias del servidor** — mantenete al tanto de novedades directamente desde el launcher
- ⚙️ **Panel de configuración** — ajustá RAM, resolución, Java y más sin complicaciones
- 🌑 **Diseño minimalista** — interfaz oscura inspirada en Modrinth y Eufonia Launcher
- ☁️ **Mods desde la nube** — los archivos se sirven desde Cloudflare R2 para descargas rápidas y confiables

## Instalación

> Por el momento el launcher está en desarrollo activo. No hay una versión pública disponible aún.

Cuando esté disponible, podés descargar el instalador desde la sección [Releases](../../releases) de este repositorio.

---

## Stack técnico

- [Electron](https://www.electronjs.org/) + Node.js
- Backend propio con [Express](https://expressjs.com/)
- [minecraft-java-core](https://github.com/luuxis/minecraft-java-core) para el lanzamiento del juego
- Archivos e instancias servidos desde [Cloudflare R2](https://www.cloudflare.com/developer-platform/r2/)
